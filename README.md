# TransitON

부산 버스·지하철 통합 데이터를 활용하여 사용자의 현재 위치를 기준으로 최적의 이동 경로와 실시간 교통 정보를 제공하는 스마트 교통 안내 서비스입니다.

## 주요 기능

* 현재 위치 기반 주변 정류장·지하철역 탐색
* 버스·지하철 실시간 도착 정보 조회
* 최적 이동 수단 자동 추천
* 예상 도착 시간 안내
* 막차 골든타임 계산 및 귀가 가이드 제공
* 즐겨찾기 경로 저장
* 최근 검색 기록 확인

## 기술 스택

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Python

### API

* 부산 버스정보관리시스템(BIMS) API
* 부산교통공사(Humetro) API
* Kakao Local API

### Database

* Supabase

### Hosting

* Vercel

## 화면 구성

1. 메인 화면
2. 위치 입력 및 검색 화면
3. 실시간 교통 정보 화면
4. 경로 추천 결과 화면
5. 막차 골든타임 안내 화면
6. 즐겨찾기 및 최근 검색 기록 화면

## 운영 흐름

사용자 위치 확인
↓
목적지 검색
↓
실시간 교통 데이터 수집
↓
최적 이동 수단 분석
↓
경로 및 도착 정보 제공
↓
즐겨찾기 저장 또는 최근 검색 기록 저장
↓
막차 이용 시 골든타임 안내

## 향후 확장 기능

* 카카오 알림 서비스 연동
* AI 기반 맞춤 이동 경로 추천
